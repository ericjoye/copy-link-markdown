# Pricing — Copy as Markdown Link

## Recommended Model: Free (with optional donation)

The extension is small, has no backend, and costs nothing to operate. There's no natural paid tier.

### Free tier (all features)
- Toolbar click copy
- Context menu (page + link)
- Badge feedback
- Title cleanup
- All future updates

### Optional monetization paths (future)
1. **"Buy me a coffee" / donation link** — one-time, voluntary. No paywall.
2. **Pro variant** (if scope expands later): bulk copy, custom templates, sync across devices. Hard paywall only if meaningful features justify it.

### Do NOT charge for this MVP
The extension is 5KB with zero server costs. Charging for it would hurt adoption and invite negative reviews. The real value is portfolio proof — showing the team can ship end-to-end (build → QA → launch).

## Revenue expectation
$0 at launch. Potential $50-200/mo via donations if the extension gets traction on GitHub/Chrome Web Store. The primary ROI is portfolio credibility, not revenue.
