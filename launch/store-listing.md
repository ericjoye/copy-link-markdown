# Chrome Web Store Listing — Copy as Markdown Link

## Extension Name

Copy as Markdown Link

## Short Description (132 chars max)

Copy any page title + URL as a `[title](url)` Markdown link with one click. Right-click or toolbar icon. 5KB. Zero deps.

## Long Description

Tired of manually typing out Markdown links?

Copy as Markdown Link is a tiny, zero-dependency Chrome extension that formats the current page's title and URL into `[title](url)` and copies it to your clipboard — instantly.

**How it works:**
- Click the toolbar icon to copy the active page as a Markdown link.
- Right-click any page → "Copy as Markdown Link".
- Right-click any hyperlink → "Copy as Markdown Link" uses the page title + link URL.

**Features:**
- ✅ One-click toolbar copy
- ✅ Right-click context menu (page + link)
- ✅ Green "Copied!" badge confirmation
- ✅ Smart title cleanup (strips " - YouTube", " | Medium", etc.)
- ✅ Link context menu uses destination URL with page title
- ✅ Works on every website

**Privacy:**
This extension makes zero network requests. No data is collected, stored, or transmitted. It only reads the current tab's title and URL to format them into Markdown.

**Permissions explained:**
- `activeTab` — read the current tab's title and URL
- `contextMenus` — add right-click menu items
- `clipboardWrite` — write the formatted link to your clipboard

**Tech:** Manifest V3 · 5KB · Zero dependencies · Open source
