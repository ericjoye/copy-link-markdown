# Copy as Markdown Link — Landing Page

## Headline

### Copy links as `[title](url)` — one click, zero fuss.

## Subhead

A 5KB Chrome extension that instantly copies any page title + URL as a Markdown link. Right from your toolbar or context menu. No bloat, no signups, no tracking.

## Benefits

- **One-click copy** — Click the toolbar icon and `[title](url)` is already on your clipboard. No more juggling tabs and manual `[` `](` keystrokes.
- **Everywhere Markdown lives** — Works instantly in Obsidian, GitHub, Slack, Notion, email, docs, and anywhere else you paste Markdown links.
- **Lightweight & private** — Zero dependencies, zero network calls, zero tracking. Just 5KB of pure Chrome API.
