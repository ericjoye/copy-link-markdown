Features

Everything you need to copy Markdown links, nothing you don't.

🖱️

One-Click Toolbar Button

Click the extension icon on any page to instantly copy [title](url) to your clipboard.

📋

Context Menu

Right-click anywhere → "Copy as Markdown link" or right-click a specific link to copy that URL.

✨

Title Cleanup

Automatically strips common site suffixes like " - YouTube", " | Medium" for clean link text.

🔔

Visual Feedback

Badge flashes "Copied!" on the toolbar icon for 1.5 seconds so you know it worked.

🔒

Zero Dependencies

Pure Chrome APIs, ~60 lines of JavaScript. No frameworks, no build step, no tracking.

How It Works

Three ways to copy a Markdown link.

1

Toolbar Click

Click the extension icon on any page. Done.

2

Context Menu

Right-click → "Copy as Markdown link" on any page.

3

Link Context

Right-click a specific link → "Copy link as Markdown".

Installation

Load in developer mode — takes 30 seconds.

1
Open Extensions
Go to chrome://extensions/ in Chrome.
2
Enable Developer Mode
Toggle the switch in the top-right corner.
3
Load Unpacked
Click "Load unpacked" and select the copy-link-markdown/ folder.
4
Done!
The icon appears in your toolbar. Start copying Markdown links.
