# Privacy Policy — Copy as Markdown Link

**Last updated:** June 20, 2026

## Overview

Copy as Markdown Link ("we", "our", "the extension") is a Chrome extension that copies the current page title and URL as a Markdown-formatted link with one click.

## Data Collection

**Copy as Markdown Link does NOT collect, store, or transmit any personal data.**

All processing happens locally in your browser. When you use the extension:

- **No data is sent to external servers.** The extension reads the page title and URL from the active tab and formats them as Markdown. Nothing is uploaded anywhere.
- **No tracking or analytics.** We do not use Google Analytics, Mixpanel, or any other tracking service.
- **No cookies.** The extension does not set or read any cookies.
- **No account required.** There is no signup, login, or user account system.

## Permissions

The extension requests the following Chrome permissions:

- **activeTab** — To access the currently active tab when you click the extension icon. Used only to read the page title and URL.
- **contextMenus** — To add "Copy as Markdown link" options to the right-click context menu.
- **clipboardWrite** — To copy the formatted Markdown link to your clipboard.

## How It Works

When you click the extension icon or use the context menu, the extension:

1. Reads the page title and URL from the active tab.
2. Strips common site suffixes (e.g., " - YouTube", " | Medium") from the title.
3. Formats the result as `[title](url)`.
4. Copies the result to your clipboard.
5. Shows a brief "Copied!" badge on the extension icon.

No data is stored, logged, or transmitted at any point.

## Third-Party Services

This extension does not integrate with any third-party services.
