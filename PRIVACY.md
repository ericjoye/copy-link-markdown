# Privacy Policy — Copy as Markdown Link

**Last updated:** July 23, 2026

## Overview
Copy as Markdown Link ("the extension") is a Chrome extension that copies the current page title and URL as a Markdown-formatted link with one click.

## Data Collection
**Copy as Markdown Link does NOT collect, store, or transmit any personal data.**

All processing happens locally in your browser. When you use the extension:
- No data is sent to external servers
- No tracking or analytics
- No cookies
- No account required

## Permissions
- **activeTab** — To access the currently active tab when you click the extension icon
- **contextMenus** — To add "Copy as Markdown link" to the right-click context menu
- **clipboardWrite** — To copy the formatted Markdown link to your clipboard
