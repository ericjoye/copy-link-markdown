3 Ways to Copy a Markdown Link [title](url) in Chrome

July 23, 2026 · 3 min read · ← Blog

If you write Markdown — in GitHub, Obsidian, Notion, or any docs — you know the pain of manually constructing links.

The typical workflow goes: highlight the URL, copy it, go back for the page title, retype it from memory, then type []( and ) around everything. It's a tiny friction that adds up when you do it dozens of times a day.

Here are three ways to copy a Markdown-formatted [title](url) link in Chrome, from fastest to simplest.

Method 1: Chrome Extension (1 click, fastest)

The Copy as Markdown Link extension adds a toolbar button that copies the current page as a Markdown link with one click. It also adds a right-click context menu option — right-click anywhere and choose "Copy as Markdown link," or right-click a specific link and choose "Copy link as Markdown."

Features:

  • One click: Click the toolbar icon → [title](url) is on your clipboard
  • Right-click support: Context menu on any page or specific link
  • Title cleanup: Automatically strips " - YouTube", " | Medium", " — GitHub" etc.
  • Visual feedback: Badge flashes "Copied!" for 1.5 seconds
  • Zero dependencies: 5KB, vanilla JS, no network calls, no tracking

Method 2: Bookmarklet (no install required)

If you can't install extensions, a bookmarklet is the next best thing. Drag this link to your bookmarks bar, then click it on any page to copy a Markdown link:

Copy as Markdown — drag to bookmarks bar:

javascript:(function(){var t=document.title.replace(/ - [^-]+$| \| [^|]+$| — [^—]+$/g,'').trim();navigator.clipboard.writeText('['+t+']('+location.href+')');alert('Copied: ['+t+']('+location.href+')')})();

This mini bookmarklet reads the page title (cleaned), grabs the URL, formats it as [title](url), and writes it to your clipboard. Works in any Chromium browser without any installation.

Method 3: Manual (no tools, basic browser skills)

The manual method takes about 10 seconds:

  1. Select the page title text
  2. Copy it (Ctrl+C or Cmd+C)
  3. Type [
  4. Paste the title (Ctrl+V)
  5. Type ](
  6. Copy the URL from the address bar
  7. Paste the URL
  8. Type )
  9. Copy the whole result

It works, but it's tedious. If you do this more than 5 times a day, the extension will save you hours over a year.

Why Markdown Links Matter

Markdown links are the standard for cross-referencing in developer documentation. GitHub READMEs, Stack Overflow answers, Notion pages, Obsidian notes, and Slack messages all use [text](url) syntax. Having a one-click way to produce properly formatted links makes writing documentation and taking research notes significantly faster.
