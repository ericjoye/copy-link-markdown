Copy as Markdown Link — One-Click Chrome Extension

July 9, 2026 · 4 min read

If you write in Markdown, copying a useful link usually means juggling tabs, copying the URL, going back for the title, and manually typing brackets. Copy as Markdown Link removes that friction: one click gives you [title](url), already on your clipboard.

Why a tiny extension still matters

The action itself is small, but it repeats constantly — in notes, docs, issues, pull requests, Slack, and email. System-wide shortcuts and native copy-link commands usually return the raw URL, not Markdown. A focused extension that does exactly one thing can still save minutes every day without adding bloat.

What it does

  • Toolbar copy — click the extension icon to copy the current page as a Markdown link
  • Context menu — right-click a page or link and choose Copy as Markdown Link
  • Badge feedback — green Copied! confirmation on the icon
  • Title cleanup — strips common suffixes like - YouTube and | Medium

Where it fits in your workflow

It works anywhere you paste Markdown: Obsidian vaults, GitHub issues, Notion docs, developer READMEs, support replies, and research notes. Because it uses standard clipboard text, there is no import/export step or app lock-in.

Honest limits

This extension does not sync links, batch-copy pages, or store history. It is intentionally simple: one formatted link, one click, no tracking, and no account required.
